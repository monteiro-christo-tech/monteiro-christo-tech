O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência visual de páginas web. Com o CSS, você pode aplicar regras de formatação, como cores, fontes e layout, separando o conteúdo (HTML) da sua representação visual. Para aprender CSS, você pode acessar tutoriais e cursos online, como os oferecidos pelo W3Schools e MDN, que fornecem exemplos práticos e explicações detalhadas sobre seletores, propriedades e princípios básicos de layout.

Para que serve?

O CSS (Cascading Style Sheets) é uma linguagem usada para definir a aparência e o layout de páginas web. Ele separa o conteúdo (HTML) da parte visual, permitindo controlar cores, fontes, tamanhos e organização dos elementos.

Principais funções do CSS:

• Estilizar elementos:
- Definir cores de texto e fundo
- Alterar fontes e tamanhos
- Aplicar bordas e efeitos visuais

• Controlar o layout:
- Posicionar elementos na página
- Criar colunas e organizar o conteúdo
- Ajustar espaçamentos

• Responsividade:
- Adaptar o site para celular, tablet e computador
- Ajustar o layout conforme o tamanho da tela

• Reutilização:
- Um único CSS pode ser usado em várias páginas
- Facilita mudanças no design

Exemplo simples de CSS:

body {
background-color: #f0f0f0;
font-family: Arial;
color: #333;
}
h1 {
color: #0066cc;
text-align: center;
}

Esse código altera a cor de fundo, a fonte e a cor do texto, além de centralizar o título.

Seletores, Propriedades e Valores

Tipo O que é Exemplo
Seletor Escolhe o elemento p, h1, .classe
Propriedade O que será alterado color, font-size
Valor Como será alterado red, 20px

Exemplo: color: red;

Conclusão

O CSS é importante porque separa o conteúdo do visual, deixando o site mais organizado.
